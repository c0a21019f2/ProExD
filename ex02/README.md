## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：＋以外の四則演算をする
- クリア(C)：entryに入力されている数字，数式を左から1文字づつをdeleteする
- オールクリア(AC)：entryに入力されている数字，数式の文字列全体をdeleteする
-パーセント：entry に入力された数字を100で割った値を出力する。
-CEボタンの実装：entryに入力されている数字，数式を右から1文字づつをdeleteする
